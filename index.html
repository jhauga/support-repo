ts-filecomp

Compare files by size, creation date, and modification date — locally or remotely — from your terminal or code.

Read the Docs Try the Demo

File Size

Compare bytes between two files and see exactly which is larger and by how much.

Date Created

Determine which file was created first, useful for tracking originals vs copies.

Date Modified

Find the most recently edited file to identify the latest working version.

Local & Remote

Compare files on disk, at URLs, or mix both — just add the --remote flag.
